# ModernBlog

ModernBlog is a responsive blogging platform built using Next.js, TypeScript, Tailwind CSS, and shadcn/ui.

## Features

- Responsive Design
- Dynamic Blog Pages
- Search Functionality
- Pagination
- Featured Posts Section
- About Page
- Contact Page
- Modern UI using shadcn/ui

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

## Installation

npm install

npm run dev
